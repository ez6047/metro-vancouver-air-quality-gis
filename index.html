GIS Final Project • Metro Vancouver

PM2.5 and Potential Source Land Use at the 1 km Scale

An ArcGIS Pro analysis of whether land-cover classes that could plausibly contribute to PM2.5 are spatially associated with higher fine particulate matter concentrations across Metro Vancouver.

Course: GEOS 370 Software: ArcGIS Pro Study Area: Metro Vancouver

Project Snapshot

  • Remote-sensed PM2.5 raster surface
  • Land-cover raster reclassified into source/non-source categories
  • Thiessen polygon and tabulated-area workflow
  • OLS and GWR models used to compare global and local relationships

1. Home

Abstract

This project examines whether the percentage of land cover classified as potential PM2.5-source land use is spatially associated with elevated PM2.5 concentrations across Metro Vancouver. The analysis uses a remote-sensed PM2.5 raster and a land-cover classification raster, both processed in ArcGIS Pro. Land-cover classes were reclassified into binary source and non-source categories, while PM2.5 raster cells were converted into points and then Thiessen polygons so that surrounding land-cover composition could be summarized consistently.

The final modelling layer was created by clipping PM2.5 polygons to the reclassified land-use polygons, tabulating area by class, calculating the percentage of each polygon covered by potential source land use, and joining that value back to the PM2.5 polygons. Ordinary least squares (OLS) regression and geographically weighted regression (GWR) were then used to compare the global and local spatial relationships between potential source land use and PM2.5.

2. Introduction

Urban Air Quality and Land-Cover Patterns

Air quality is a central urban environmental issue because it influences respiratory health, cardiovascular risk, and overall quality of life. Fine particulate matter, especially PM2.5, is commonly used as an indicator because these particles are small enough to remain suspended in the air and penetrate deeply into the lungs.

Urban land-cover patterns can influence PM2.5 exposure because built surfaces, paved areas, industrial activity, transportation corridors, and other intensive land uses often cluster in particular parts of a metropolitan region. Metro Vancouver is a useful study area because it contains dense urban cores, suburban development, agricultural areas, industrial lands, water bodies, and large vegetated areas within a relatively compact region.

Why GIS for This Question?

GIS makes it possible to align raster and vector datasets, reclassify land-cover categories, summarize area by polygon, and visualize spatial patterns that would be difficult to interpret from tables alone.

3. Project Objective

Research Focus

Main Research Question

Is the percentage of potential PM2.5-source land use spatially associated with higher PM2.5 concentrations across Metro Vancouver?

Project Goals

  • Reclassify the land-cover raster into potential PM2.5 source and non-source categories.
  • Convert PM2.5 raster cells into Thiessen polygons for area-based comparison.
  • Calculate the percentage of each PM2.5 polygon covered by potential source land use.
  • Use OLS and GWR to compare global and local relationships between land-use composition and PM2.5.

4. Data

Input Datasets

PM2.5 Raster

PM2.5 refers to fine particulate matter with particle diameters less than 2.5 micrometres. The PM2.5 input is a remote-sensed raster dataset sourced from researchers at Washington University in St. Louis. It represents yearly mean PM2.5 concentration and has approximately 0.01° resolution, with grid cells roughly 1 km in length. PM2.5 values are reported in µg/m³.

5. Methodology

ArcGIS Workflow

Workflow Design

The two main inputs were not directly comparable in their original forms. The land-cover dataset was a categorical raster, while the PM2.5 surface was a continuous raster of yearly mean concentration values. The workflow therefore converted both datasets into a shared polygon-based analytical structure so that each PM2.5 zone could be assigned a corresponding percentage of land classified as a potential PM2.5 source. The final output was a polygon layer in which each feature contained both a PM2.5 value and an explanatory variable representing the proportion of nearby land use coded as a potential source.

This workflow was designed to preserve the original PM2.5 grid values while translating the land-cover classification into a simpler binary variable suitable for regression analysis. Each geoprocessing step was used to move from raw raster inputs toward a final modelling layer that could support OLS regression, GWR, and residual spatial autocorrelation testing.

01

Define the Explanatory Variable

The original land-cover classes were collapsed into a binary source/non-source classification so the predictor variable represented the percentage of land in each PM2.5 zone that could plausibly contribute to particulate pollution.

02

Create Comparable Spatial Units

The PM2.5 raster was converted into Thiessen polygons so each PM2.5 value could be treated as a polygon-based analysis unit for area calculations and later regression modelling.

03

Summarize Source Area

Overlay tools were used to measure how much source-coded and non-source-coded land occurred within each PM2.5 polygon, allowing land-cover composition to be summarized numerically.

04

Build the Regression Layer

The calculated percentage of potential source land use was joined back to the PM2.5 polygons, producing the final layer used in OLS and GWR analysis.

Detailed Step-by-Step Methods

  1. Step 1: Reclassify the land-use raster. The original land-cover raster contained multiple classes such as built surfaces, vegetation types, water, and other cover categories. These classes were reclassified into a binary field using the criteria table shown in the methods figure. A value of 0 represented land-cover classes not treated as potential PM2.5 sources, while a value of 1 represented classes treated as potential PM2.5 source land use. This step reduced a complex categorical dataset into a simpler explanatory variable that could be measured consistently across the study area.
  2. Step 2: Convert the reclassified raster to polygons. After reclassification, the binary raster was converted to polygon features. This made it possible to calculate land area by class and overlay the land-use data with polygon-based PM2.5 analysis units in later steps.
  3. Step 3: Convert the PM2.5 raster to points. Each raster cell was converted into a point feature carrying its PM2.5 concentration value. This preserved the original concentration surface while preparing the data for Thiessen polygon generation.
  4. Step 4: Convert PM2.5 points to Thiessen polygons. Thiessen polygons were created so that each PM2.5 point defined the surrounding area closest to that value. This produced a continuous polygon tessellation that could be intersected with the reclassified land-use polygons. The Thiessen approach converted the raster-based PM2.5 surface into discrete areal units without averaging neighbouring cells.
  5. Step 5: Clip PM2.5 polygons to the land-use polygon layer. The PM2.5 Thiessen polygons were clipped using the reclassified land-use polygon extent so both datasets occupied the same spatial footprint. This ensured that area summaries were calculated only where both PM2.5 and land-use information existed.
  6. Step 6: Use Tabulate Area to summarize land-cover area inside each PM2.5 polygon. The Tabulate Area tool calculated the total area of land coded as non-source and the total area coded as potential source within each PM2.5 polygon. This transformed the binary land-use classification into numeric area totals for each analysis unit.
  7. Step 7: Use Calculate Field to compute the percentage of potential source land use. The source-area totals from Tabulate Area were converted into a percentage variable representing the proportion of each PM2.5 polygon classified as potential PM2.5 source land use. This percentage became the main explanatory variable used in regression modelling.
  8. Step 8: Join the tabulated results back to the clipped PM2.5 polygons. The calculated area and percentage fields were joined back to the clipped PM2.5 polygon layer. The resulting feature class contained both the PM2.5 value and the percentage of potential source land use for each polygon, making it suitable for mapping, OLS regression, GWR, and residual diagnostics.

Percent potential source land use = (area coded as potential PM2.5 source / total land-use area in the polygon) × 100

6. Results

Maps and Model Outputs

OLS Model Interpretation

Global R² = 0.201

The percent potential-source land-use variable explains approximately 20.1% of the observed variation in PM2.5 across the analysis polygons.

Coefficient = 0.018248

Each 1 percentage point increase in potential PM2.5-source land use is associated with a 0.018248 µg/m³ increase in PM2.5.

The OLS model shows a positive global relationship between potential PM2.5-source land use and PM2.5 concentration. An R² of 0.201 means that the land-use percentage variable explains about one-fifth of the variation in PM2.5 values. This is a meaningful but limited relationship: most PM2.5 variation remains unexplained by this single predictor, which suggests that other factors such as meteorology, topography, traffic volume, industrial emissions, regional background pollution, and atmospheric transport also influence the observed PM2.5 surface.

The coefficient of 0.018248 means that, on average, a 1 percentage point increase in land use classified as a potential PM2.5 source corresponds to an increase of 0.018248 µg/m³ in PM2.5. Put another way, a 10 percentage point increase in potential source land use corresponds to an expected increase of approximately 0.18248 µg/m³. This coefficient should be interpreted as an association, not proof that the land-use classes directly caused the measured PM2.5 values.

Residual Spatial Autocorrelation

The OLS residuals show very strong positive spatial autocorrelation. The Global Moran’s I value is 0.957019, with a z-score of 76.098159 and a p-value reported as 0.000000. This means the OLS residuals are highly clustered rather than randomly distributed, so the global regression model does not fully account for the spatial structure in PM2.5.

The GWR residuals also remain spatially autocorrelated, but less strongly than the OLS residuals. The Global Moran’s I value decreases to 0.678473, with a z-score of 54.015470 and a p-value reported as 0.000000. This suggests that GWR captures some of the spatial non-stationarity in the relationship, but substantial spatial structure remains unexplained.

7. Discussion

Interpretation and Limitations

The analysis suggests a positive global association between potential PM2.5-source land use and PM2.5 concentration, but the relationship is not strong enough to explain the full spatial pattern on its own. The OLS R² of 0.201 indicates that land-use composition explains about 20.1% of PM2.5 variation, while the strong Moran’s I value for OLS residuals shows that substantial spatial structure remains unexplained.

GWR improves the interpretation by allowing the relationship to vary across space. The local model-fit map shows that the explanatory strength of the land-use variable is not constant across Metro Vancouver. This supports the use of a spatially varying model, but the remaining GWR residual autocorrelation shows that land use alone is still not sufficient to explain PM2.5 patterns.

The results should be interpreted cautiously. The land-cover variable is a proxy: it estimates the percentage of each PM2.5 polygon occupied by land uses that could plausibly contribute to fine particulate matter, but it does not directly measure emissions. It also does not capture traffic counts, industrial emission rates, fuel type, wind, topography, temperature, atmospheric mixing, or regional background pollution.

A second limitation is the spatial resolution of the PM2.5 raster. The approximately 1 km scale is appropriate for regional comparison but may smooth fine-scale pollution gradients near roads, industrial parcels, or other localized sources. The workflow is also affected by classification uncertainty, edge effects from Thiessen polygons, and the modifiable areal unit problem because results may change under different polygon boundaries or raster resolutions.

Future Extensions

  • Use direct emissions, traffic count, or industrial facility data instead of only land-cover proxies.
  • Add meteorological and topographic variables to account for dispersion and accumulation.
  • Test multiple spatial resolutions to evaluate scale sensitivity and MAUP effects.
  • Compare results using alternative land-cover reclassification rules.
  • Use a multivariate spatial model if additional explanatory variables are available.
