# GEOS 370 Final Project Website

This repository includes a simple single-page website built with plain `HTML`, `CSS`, and `JavaScript` so it can be hosted directly on GitHub Pages without a build step.

## Files

- `index.html` - main page structure and project content
- `style.css` - layout, colors, responsive design, and visual styling
- `script.js` - mobile navigation, scroll reveal, and back-to-top button

## How to publish on GitHub Pages

1. Create a GitHub repository and upload these files to the repository root.
2. Commit and push the repository to GitHub.
3. In the repository, open `Settings` → `Pages`.
4. Under `Build and deployment`, set `Source` to `Deploy from a branch`.
5. Select the branch you want to publish, usually `main`, and keep the folder as `/ (root)`.
6. Save the settings and wait for GitHub Pages to publish the site.
7. After deployment, GitHub will provide a public Pages URL for the website.

## Editing notes

- Replace the placeholder text in `index.html` with your final project writing.
- Replace the map and chart placeholder boxes with exported ArcGIS images.
- Update the footer, title, subtitle, course details, and author information as needed.
- If you add real images, place them in an `images/` folder and update the `<img>` or placeholder sections in `index.html`.

## Local preview

You can preview the site by opening `index.html` directly in a browser.
